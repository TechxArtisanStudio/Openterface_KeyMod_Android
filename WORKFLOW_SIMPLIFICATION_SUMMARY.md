## 🚀 For Production Releases (Future)

### When Ready for Play Store:

**1. Generate Release Keystore:**
```bash
keytool -genkey -v -keystore keymod-release.jks \
  -alias keymod \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -storepass <password> \
  -keypass <password> \
  -dname "CN=TechxArtisan, OU=Development, O=TechxArtisanStudio, L=Shanghai, S=Shanghai, C=CN"
```

**2. Store Securely:**
```bash
# NEVER commit keystore to Git
# Store in secure location:
# - Company vault
# - Password manager
# - Encrypted backup
```

**3. Add GitHub Secrets:**
```
Settings → Secrets and variables → Actions

SIGNING_KEY=[base64 -w 0 keymod-release.jks]
KEY_STORE_PASSWORD=<store password>
KEY_PASSWORD=<key password>
```

**4. Update Workflow:**
```yaml
- name: Decode Keystore
  run: |
    echo $SIGNING_KEY | base64 -di > app/keystore.jks

- name: Build Release APK
  run: ./gradlew assembleRelease \
    -PRELEASE_STORE_FILE=keystore.jks \
    -PRELEASE_STORE_PASSWORD=$KEY_STORE_PASSWORD \
    -PRELEASE_KEY_ALIAS=keymod \
    -PRELEASE_KEY_PASSWORD=$KEY_PASSWORD

- name: Upload Release APK
  uses: actions/upload-artifact@v4
  with:
    name: KeyMod-release
    path: app/build/outputs/apk/release/*.apk
```
