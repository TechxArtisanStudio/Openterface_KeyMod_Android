**Visual:** Split screen showing both orientations side by side

## 🎥 Production Notes

### Recording Method:
**Option 1: Screen Recording (Recommended)**
- Use Android device or emulator
- Record with scrcpy or Android Studio screen recorder
- Rotate device during recording
- Capture smooth transitions

**Option 2: Mockup Animation**
- Create static screenshots of both orientations
- Animate rotation transition
- Add text overlays and effects
- Professional look, easier to produce

### Tools Needed:
- **Recording:** scrcpy, Android Studio, or physical device
- **Editing:** DaVinci Resolve, Shotcut, or OpenShot
- **Graphics:** GIMP, Inkscape, or Canva
- **Audio:** Audacity (for music/sound effects)

### Export Settings:
- **Resolution:** 1920×1080 (Full HD)
- **Format:** MP4 (H.264)
- **Frame Rate:** 30 fps
- **Bitrate:** 8-10 Mbps
- **Audio:** AAC, 128 kbps

## 📱 Alternative: Quick Demo (5 minutes)

If full video production is too time-consuming, create a simple screen recording:

1. **Connect Android device** via USB
2. **Start screen recording** with scrcpy:
   ```bash
   scrcpy --record orientation-demo.mp4
   ```
3. **Demonstrate:**
   - Open app in portrait
   - Rotate to landscape
   - Show both layouts
   - Navigate to settings
   - Show rotation works everywhere
4. **Stop recording**
5. **Upload** to YouTube/GitHub
